# Math Utils

Simple math utilities package for demonstration of **CI/CD with GitHub Actions**.

![CI](https://github.com/FranzCastillo/CI-CD-Basic-Math-Package/actions/workflows/ci.yml/badge.svg)

## Functions
- `square(n)`
- `factorial(n)`
- `is_prime(n)`
- `gcd(a, b)`
- `lcm(a, b)`

## Install
```bash
pip install .
```
